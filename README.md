# Customer Segmentation using K-Means Clustering

##  Description
This project uses **K-Means Clustering** to segment customers based on their **annual income** and **spending score**. It helps identify different customer groups for better marketing strategies.

---

## 📖 Project Overview
- Applied **unsupervised learning (K-Means)**
- Used **Elbow Method** to determine optimal clusters
- Visualized customer segments using scatter plots

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

##  Dataset
- Mall Customer Dataset  
- Features used:
  - Annual Income (k$)
  - Spending Score (1-100)
